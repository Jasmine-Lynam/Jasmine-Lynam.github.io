Final Year Project - Ongoing

Description

In recent months rather than being on campus, students are returning to college remotely. For a number of students this raises an issue, as the vast majority of students are using a budget line laptop that is lacking the technical requirements needed for their coursework. This scenario applies to a number of technical students as well as those with outdated machines.The concept for my project is to implement a Virtual Desktop Infrastructure for students working remotely from home. Here, by incorporating a VDI solution there is no need for students to purchase a top of the range desktop. Throughout the course of my project I’ll be comparing various VDI solutions and determining which one is best suited in a remote academic environment.

Technologies & Tools Used

Azure, Amazon WorkSpaces, Citrix Cloud
